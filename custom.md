# Instructions

## Overview

In this tutorial we will explore the principles of Video Game Programming and Video Game Design by creating a version of the the space shooter game GALGA. Click NEXT to get started!

## Events

An event is something that happens that makes code run.

Our first **event** is the start of the game.

Anything placed in the ``||loops: on start||`` block will initialize at the beginning of the program.

**FROM the GAME section of the tool bar, find the ``||game: game.splash("")||`` block and place it inside the ``||loops:on start||`` loop. ``||game: game splash||`` gives you a Menu screen.  This screen can feature the games title and instructions on how to play. When players press the A button, the game  will begin.  Type the name of your game inside the oval in the ``||game:game.splash("")||`` block. **

```blocks
game.splash("GALGA!")
```

## Sprites

Let's create the main player controlled sprite. In the ``||Sprites:Sprites||`` menu select the ``||Variables:set mySprite||`` ``||Sprites:to sprite of kind player||`` block and place it below the game splash block:
